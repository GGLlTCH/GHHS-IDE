# GHHS-IDE
GHHS IDE - это мобильное приложение, разработанное на Kotlin, представляющее собой удобный и компактный редактор кода, предназначенный для оперативного редактирования и внесения небольших изменений в существующие проекты на Python, C++, JavaScript (JS), React и Java. GHHS IDE не является заменой полноценной настольной IDE для разработки
